# 試験タイマー

宅建士試験（50問・120分）を **1問2分ペース**で進めるための、ブラウザ完結の学習用タイマー。

- 現在「第N問 / 50」を大きく表示
- 「この問題の残り（目安2分）」をカウントダウン（残り少→黄、超過→赤点滅＋ビープ）
- 全体の残り時間とペース（予定より ＋/− 何分何秒）を可視化
- **途中の問題から開始**できる（選んだ問題以降の持ち時間に残り時間を自動調整）
- 問題数・制限時間・1問の目安秒・終了音は設定で変更可（localStorage 保存）
- 単一 HTML・ビルド不要・1画面に収まる（スクロール無し）

## 使い方

`index.html` をブラウザで開くだけ。

| 操作 | 動作 |
|---|---|
| `Space` | スタート / 一時停止 / 再開 |
| `→` または `Enter` | 次の問題へ |
| `←` | 前の問題へ |
